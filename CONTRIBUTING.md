# Guia de Contribuição 🚀

Primeiramente, obrigado por dedicar seu tempo para contribuir com este projeto! São pessoas como você que tornam a comunidade GitHub incrível.

Este documento serve para orientar o processo de contribuição, desde o reporte de bugs até o envio de novas imagens ou informações sobre badges.

---

## 🧐 Como posso contribuir?

### 1. Relatando Bugs ou Informações Incorretas

Se você encontrou um erro em uma descrição, um link quebrado ou uma imagem que não carrega:

* Verifique se já existe uma **Issue** aberta sobre o assunto.
* Caso não exista, abra uma nova Issue descrevendo o problema detalhadamente.

### 2. Sugerindo Melhorias

Tem uma ideia para organizar melhor a lista ou encontrou uma nova badge que ainda não listamos?

* Abra uma Issue com a tag `enhancement` (melhoria).
* Se você mesmo quiser implementar, siga os passos de **Pull Request** abaixo.

### 3. Enviando Pull Requests (PRs)

Para contribuições diretas no código ou conteúdo:

1. Faça um **Fork** do repositório.
2. Crie uma **Branch** para sua modificação:
```bash
git checkout -b feature/nova-conquista

```


3. Faça o **Commit** das suas alterações com mensagens claras:
```bash
git commit -m "Adiciona detalhes sobre a conquista X"

```


4. Faça o **Push** para a sua branch:
```bash
git push origin feature/nova-conquista

```


5. Abra um **Pull Request** para a branch `main` deste repositório.

---

## 🎨 Padrões de Conteúdo

Para manter a organização do guia, pedimos que siga estes padrões:

* **Imagens:** Use os arquivos na pasta `src/assets/badges/`. Prefira formatos leves (PNG ou SVG).
* **Tabelas:** Mantenha o alinhamento centralizado nas tabelas do Markdown para facilitar a leitura.
* **Idioma:** O conteúdo principal deve seguir o padrão definido (se o README for em português, mantenha o português).
* **Links:** Sempre teste os links para a documentação oficial do GitHub antes de enviar.

## ⚖️ Código de Conduta

Ao interagir com este projeto, você concorda em manter um ambiente respeitoso, colaborativo e livre de assédio. Tratamos todos com gentileza, independentemente do nível de experiência.

---
